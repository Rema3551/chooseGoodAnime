# ChooseGoodAnime

Le futur de la recommandation d'animes.

## Prérequis
- **Node.js** (version 18 ou supérieure recommandée)
- npm (installé avec Node.js)

## Installation & Lancement

Ce projet nécessite Node.js pour fonctionner. L'environnement actuel n'ayant pas détecté Node.js, vous devrez peut-être l'installer ou l'ajouter à votre PATH si ce n'est pas déjà fait.

1.  Ouvrez un terminal dans ce dossier.
2.  Installez les dépendances :
    ```bash
    npm install
    ```
3.  Lancez le serveur de développement :
    ```bash
    npm run dev
    ```
4.  Ouvrez votre navigateur sur l'URL indiquée (généralement `http://localhost:5173`).

## Structure du Projet
- `src/App.jsx` : Le code principal de l'application (Interface).
- `src/index.css` : Le design system et les styles globaux.
- `src/data/` (À venir) : Les données factices pour tester les filtres.
